# Lumiere - Fashion Muslim Modern
A modern and elegant Muslim fashion landing page built with Next.js and Tailwind CSS.
## Live Demo
https://lumiere-fashion-eta.vercel.app
## Features
- Hero section with parallax effect
- Product collection with category filter
- About section with scroll reveal animations
- Customer testimonials carousel
- Contact form with WhatsApp integration
- Fully responsive mobile-first design
## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Pure CSS animations
## Getting Started
`ash
npm install
npm run dev
`
## Reusable Template
Easily adaptable for any fashion or retail business by changing colors, content, and images.
